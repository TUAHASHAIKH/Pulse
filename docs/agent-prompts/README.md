# Agent Prompts

This directory contains the versioned system prompts for each AI agent.
Prompts define agent behaviour as much as code does — keep them under version control.

## Structure (Phase 2+)

```
agent-prompts/
├── architect_agent.md
├── security_agent.md
├── performance_agent.md
├── code_quality_agent.md
└── repair_agent.md
```

Each file contains the system prompt sent to the LLM when that agent is invoked.
